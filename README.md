# Sistema Venda Fácil 

Projeto Full Stack de um sistema de gerenciamento de vendas, desenvolvido para fins de aprendizado e testes de integração entre Backend e Frontend.

## Tecnologias Utilizadas

* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** Node.js, Express
* **Banco de Dados:** SQLite (venda-facil.db)

## Estrutura do Projeto

* `/backend`: API REST construída em Node.js com rotas para produtos, clientes e vendas.
* `/frontend`: Interface do usuário simples e intuitiva.

## Como rodar o projeto localmente

1. **Clone o repositório:**
   </> bash
   git clone [https://github.com/Allax-Dev/sistema-vendaafacil-teste.git](https://github.com/Allax-Dev/sistema-vendaafacil-teste.git)

2. **Acesse a pasta do backend** 
   </> bash 
    cd sistema-vendaafacil-teste/backend

3. **Instale as dependências**
   </> bash
   npm install

4. **Inicie o servidor**
   </> bash
   node server.js
   A API estará rodando em http://localhost:3000

5. **Abrir o Frontend**
  Vá até a pasta /frontend 
  Abra o arquivo index.html no navegador

**Observações**
O banco de dados SQLite (venda-facil.db) é criado automaticamente
Projeto desenvolvido com foco em organização e boas práticas
Ideal para estudos de integração entre frontend e backend
